# Оркестрирование задач обработки данных: Контроль домашних заданий в школе

## Лабораторная работа №2 по курсу «Технологии Big Data»

### Цель работы
Научиться создавать системы работы с потоками данных с использованием инструмента Apache Airflow.

### Решаемые задачи
- Развёртывание Apache Airflow в среде WSL2 (Windows Subsystem for Linux)
- Разработка DAG (Directed Acyclic Graph) для симуляции процесса контроля домашних заданий
- Оркестрирование выполнения задач: генерация данных, симуляция сдачи работ, расчёт статистики
- Загрузка DAG в Apache Airflow и настройка автоматического запуска по расписанию

### Отражаемые сущности
- **Ученики** (id, имя, класс)
- **Учителя** (ФИО, предмет)
- **Предметы** (математика, русский язык, информатика, история, биология)
- **Задания** (id, предмет, описание, срок сдачи)
- **Сданные работы** (факт сдачи, дата, статус)
- **Оценки** (от 2 до 5, генерируются случайно с заданными весами)

### Структура DAG (`school_homework_control`)
DAG состоит из следующих задач (PythonOperator и BashOperator):

| Task ID | Описание |
|---------|----------|
| `generate_students` | Генерация списка учеников |
| `generate_teachers_subjects` | Генерация учителей и закреплённых предметов |
| `generate_assignments` | Генерация домашних заданий |
| `simulate_submissions` | Симуляция сдачи работ (вероятность 80%) и выставление оценок |
| `save_results_to_csv` | Сохранение результатов в CSV-файл |
| `calculate_statistics` | Расчёт статистики (общее количество, средний балл, доля сдавших) и сохранение в JSON |
| `echo_completion` | Информационное сообщение об успешном завершении |


### Расписание выполнения
DAG настроен на автоматический запуск каждый день в **18:00** (cron-выражение `0 18 * * *`).

### Результаты работы
После успешного выполнения DAG в папке `data/` создаются:
- `results_YYYYMMDD_HHMMSS.csv` – детальная таблица сданных работ и оценок
- `stats_YYYYMMDD_HHMMSS.json` – сводная статистика

Пример содержимого CSV:
| student_id | student_name      | subject       | status | grade |
|------------|-------------------|---------------|--------|-------|
| 1          | Анна Иванова      | Математика    | Сдано  | 4     |
| 2          | Борис Петров      | Информатика   | Не сдано | NULL |
